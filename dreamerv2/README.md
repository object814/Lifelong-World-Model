# DreamerV2 Implementation on DeepMind Control Suite Custom Walker Environments.
This repository contains an implementation of the DreamerV2 algorithm applied to custom Walker environments from the DeepMind Control Suite. The implementation is based on the original DreamerV2 paper by Hafner et al. and adapted to work with the Walker tasks.
